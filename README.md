# ANODA-Turn-Timer
ANODA Open Source iOS Swift example app

[http://itunes.apple.com/app/id1282215925]()

## Build

Building ANODA Turn Timer requres Xcode 9+ and CocoaPods.

Install pods

```
pod install
```

Then just build it in your Xcode
